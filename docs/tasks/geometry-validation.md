# 几何校核模块任务

模块目标：提供独立几何校核工具，检查非法非端点交叉、连接点间距、核心筒保护区侵入，并输出可定位的问题信息。

对应需求：F3、F4、F5、F6、F7、F10、N2。

相关文件：

- 修改：`strut_validation.py`
- 关联：`strut_engine.py`
- 验证：`test_strut_minimal.py`

## 最小任务清单

- [ ] 任务 1：修复 `strut_validation.py` 中缺失的 `is_connection_point()`。
- [ ] 任务 2：定义校核输入结构，支持读取旧字段 `struts`、`corners`、`ties` 和新字段 `members`。
- [ ] 任务 3：实现构件线段收集函数，支持主撑、角撑、八字撑、连杆、桁架腹杆、径向撑。
- [ ] 任务 4：实现圆环撑收集函数，支持 `ring_strut` 类型。
- [ ] 任务 5：实现端点触碰判断，端点容差可配置。
- [ ] 任务 6：实现合法连接点判断，允许构件在同一节点连接。
- [ ] 任务 7：实现允许交叉规则表，例如主撑与主撑交点可作为立柱节点。
- [ ] 任务 8：实现非法非端点交叉检测。
- [ ] 任务 9：非法交叉问题输出构件类型、构件 ID 或索引、交点坐标和违规原因。
- [ ] 任务 10：实现腰梁连接点间距提取。
- [ ] 任务 11：实现连接点间距推荐范围校核，范围来自 `spacing_min` 和 `spacing_max`。
- [ ] 任务 12：实现核心筒保护区圆形区域构造，半径为 `core_diameter / 2`。
- [ ] 任务 13：实现核心筒保护区侵入校核，禁止任何支撑构件进入。
- [ ] 任务 14：实现圆环内侧空置校核，禁止主撑、径向撑、斜撑、腹杆、连杆、调试线穿过圆环内部。
- [ ] 任务 15：将校核结果写入 `issues`。
- [ ] 任务 16：将长度统计入口预留给统计模块或调用统计模块。
- [ ] 任务 17：为矩形 60m x 40m 写非法角撑/主撑交叉回归用例。
- [ ] 任务 18：为连杆互穿写非法交叉回归用例。
- [ ] 任务 19：为核心筒保护区侵入写回归用例。
- [ ] 任务 20：为主撑与主撑合法交点写允许交叉用例。
- [ ] 任务 21：运行 `python test_strut_minimal.py`，确认失败时输出坐标和违规原因。

## 验收清单

- [ ] `strut_validation.py` 不再引用未定义函数。
- [ ] 非法非端点交叉可被报告。
- [ ] 合法节点连接不会被误报。
- [ ] 核心筒保护区侵入可被报告。
- [ ] 连接点 6m 到 9m 推荐范围可被校核。
