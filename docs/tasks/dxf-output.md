# DXF 输出模块任务

模块目标：按稳定图层输出基坑边界、腰梁、支撑中心线、圆环撑、径向撑、立柱、调试图层和后续支撑边线。

对应需求：F11、F13、N1、N3。

相关文件：

- 修改：`main_strut.py`
- 关联：`strut_engine.py`
- 验证：手动打开输出 DXF 或通过 DXF 实体读取检查图层。

## 最小任务清单

- [ ] 任务 1：梳理 `main_strut.py` 当前 DXF 写出逻辑和已有图层名。
- [ ] 任务 2：新增或确认 `WALL_REF` 图层，用于基坑边界参考。
- [ ] 任务 3：新增或确认 `WALING` 图层，用于腰梁中心线。
- [ ] 任务 4：新增或确认 `STRUT` 图层，用于主撑中心线，包含对撑形式。
- [ ] 任务 5：新增或确认 `CORNER` 图层，用于角撑/八字撑。
- [ ] 任务 6：新增或确认 `TIE` 图层，用于连杆。
- [ ] 任务 7：新增或确认 `TRUSS_WEB` 图层，用于桁架腹杆。
- [ ] 任务 8：新增或确认 `RING_STRUT` 图层，用于圆环撑。
- [ ] 任务 9：新增或确认 `RADIAL_STRUT` 图层，用于径向撑。
- [ ] 任务 10：新增或确认 `PILLAR` 图层，用于立柱。
- [ ] 任务 11：预留 `STRUT_OUTLINE` 图层，用于支撑边线或后续闭合轮廓。
- [ ] 任务 12：预留 `WALING_OUTLINE` 图层，用于腰梁边线或后续闭合轮廓。
- [ ] 任务 13：预留 `TRUSS_CHORD` 图层，用于桁架弦杆。
- [ ] 任务 14：预留 `DEBUG_NODE` 图层，用于节点调试。
- [ ] 任务 15：预留 `DEBUG_ISSUE` 图层，用于校核问题定位。
- [ ] 任务 16：预留 `CORE_PROTECTION` 图层，用于核心筒保护区。
- [ ] 任务 17：实现从旧 layout 字段绘制基础中心线，保持现有输出兼容。
- [ ] 任务 18：实现从 `members` 绘制新增构件类型，避免新增体系无法出图。
- [ ] 任务 19：实现从 `outlines` 绘制支撑边线。
- [ ] 任务 20：当前阶段允许中心线作为调试图层保留。
- [ ] 任务 21：写一个 DXF 图层读取检查，确认基础图层存在。
- [ ] 任务 22：对 `circular` 用例检查 `RING_STRUT`、`RADIAL_STRUT`、`CORE_PROTECTION` 图层存在。
- [ ] 任务 23：对 `straight_truss` 用例检查 `TRUSS_WEB` 和 `TRUSS_CHORD` 图层存在。

## 验收清单

- [ ] DXF 至少包含全部基础图层。
- [ ] 圆环撑和径向撑有独立图层。
- [ ] 桁架腹杆和弦杆可分层输出。
- [ ] 调试节点和问题可选输出。
- [ ] 支撑中心线和边线可并行输出。
