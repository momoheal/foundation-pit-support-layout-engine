# 几何生成策略模块任务

模块目标：按支撑体系生成腰梁、主撑、角撑/八字撑、连杆/腹杆、桁架、圆环撑、径向撑和立柱候选节点。

对应需求：F5、F6、F7、F8、F9、F10、N1、N5。

相关文件：

- 修改：`strut_engine.py`
- 关联：`strut_validation.py`
- 验证：`test_strut_minimal.py`

## 最小任务清单

- [ ] 任务 1：拆分现有 `_solve_rectangular()` 中的腰梁生成逻辑，形成可复用的腰梁生成步骤。
- [ ] 任务 2：确保腰梁输出为闭合折线，并保留在兼容字段 `waling`。
- [ ] 任务 3：拆分现有 `_place_struts()`，形成正交主撑生成步骤。
- [ ] 任务 4：确保主撑端点落在腰梁内部边界或合法节点上。
- [ ] 任务 5：为主撑生成节点记录，包括端点和主撑交点。
- [ ] 任务 6：在生成主撑时跳过短于 `min_strut_len` 的构件。
- [ ] 任务 7：在阳角附近保留两个方向对撑加强的候选逻辑。
- [ ] 任务 8：拆分角撑/八字撑候选生成，不直接写入最终 layout。
- [ ] 任务 9：将角撑/八字撑候选先提交几何校核，合法后再加入 `members` 和兼容字段 `corners`。
- [ ] 任务 10：当角撑/八字撑候选非法交叉时，先实现“跳过候选”的最小策略。
- [ ] 任务 11：拆分连杆生成逻辑，默认不生成双向互穿连杆网格。
- [ ] 任务 12：实现连杆端点吸附到主撑、腰梁或合法节点。
- [ ] 任务 13：将连杆生成结果写入 `members` 和兼容字段 `ties`。
- [ ] 任务 14：新增桁架生成骨架：在两条对撑之间生成桁架区域。
- [ ] 任务 15：在有主撑区域生成 Warren 型腹杆。
- [ ] 任务 16：在无主撑区域生成 K 型腹杆。
- [ ] 任务 17：将桁架节点写入统一节点表。
- [ ] 任务 18：新增圆环撑生成骨架，生成一圈真圆。
- [ ] 任务 19：根据 `core_diameter / 2 + core_clearance` 校验圆环最小半径。
- [ ] 任务 20：生成径向撑时只连接圆环外侧和外部支撑系统。
- [ ] 任务 21：确保径向撑不穿过圆环内部核心筒保护区。
- [ ] 任务 22：将环撑与径向撑交点写入统一节点表。
- [ ] 任务 23：为矩形 60m x 40m 用例验证不会生成角撑与主撑非法交叉。
- [ ] 任务 24：为 L 形用例验证凹角处优先生成角撑候选。
- [ ] 任务 25：为圆形/椭圆带核心筒用例验证圆环内部空置。
- [ ] 任务 26：直边桁架策略沿腰梁内侧生成连续或分段连续边桁架，边桁架由弦杆和三角腹杆组成。
- [ ] 任务 27：直边桁架策略中的连杆只连接相邻主撑、主桁架或合法桁架节点，并将中段连接点写入节点表。
- [ ] 任务 28：八边形或切角近圆基坑在显式选择 `circular` 时生成圆环撑、径向撑和核心筒保护区。
- [ ] 任务 29：边桁架采用腰梁作为外弦，内侧偏移生成内弦和三角腹杆。
- [ ] 任务 30：大基坑角撑生成多道角部桁架带，且不与单线角撑叠加。
- [ ] 任务 31：立柱从拓扑候选中筛选，满足 `pillar_min_spacing` 最小间距。

## 验收清单

- [ ] 正交对撑能稳定生成腰梁和主撑。
- [ ] 八字撑/角撑候选经过校核后才加入结果。
- [ ] 默认不生成双向互穿连杆网格。
- [ ] 直边桁架包含以腰梁为外弦的边桁架、内部主桁架、多三角腹杆和合法连杆。
- [ ] 大基坑角撑采用角部桁架带。
- [ ] 立柱数量和间距经过候选评分筛选，不直接等于桁架节点数。
- [ ] 内圆型支撑生成一圈真圆且保护核心筒。
