# 命令行测试模块任务

模块目标：提供不依赖 GUI 的最小回归测试脚本，覆盖典型基坑和关键几何违规。

对应需求：F4、F3、F5-F13、N2。

相关文件：

- 修改：`test_strut_minimal.py`
- 关联：`strut_engine.py`
- 关联：`strut_validation.py`

- [ ] 任务 2：定义测试用例结构，包含名称、边界点、参数、期望结果。
